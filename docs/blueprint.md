# **App Name**: NewsWave

## Core Features:

- Dynamic News Feed: Display news articles from various open-source APIs in a continuously updated feed.
- Category Filters: Enable users to filter news articles by predefined categories (e.g., sports, politics, technology).
- Search Bar: Implement a search bar that allows users to find articles on specific topics using keywords.
- Ad Placement: Strategically integrate ad placements within the news feed to monetize the platform. Each position will serve contextually appropriate ads using a third-party tool.
- Source Attribution: Clearly label each news article with its source to maintain transparency and credibility.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) for a calm, trustworthy feel.
- Background color: Very light blue (#F0F8FF) to create a clean, uncluttered backdrop.
- Accent color: Deep blue (#3F72AF) to draw attention to calls to action and important elements.
- Body and headline font: 'Inter', a sans-serif for a modern, neutral look.
- Use simple, consistent icons to represent categories and navigation elements.
- Maintain a clean and modern layout with ample white space to enhance readability and user experience.
- Subtle animations to enhance user interaction (e.g., loading indicators, hover effects).
